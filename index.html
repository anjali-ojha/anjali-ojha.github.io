Phishing Website Detection using Source Code

In this initiative, an application was developed to automatically detect phishing websites by analyzing their URL and HTML source code. The primary aim was to leverage advanced machine learning techniques to effectively distinguish malicious web pages from legitimate ones. Through comprehensive feature extraction and modeling, the application achieved an impressive accuracy rate of 96.35%, utilizing the powerful XGBoost algorithm.
This achievement not only enhanced security measures but also demonstrated the efficacy of combining domain-specific features with machine learning. Such a tool provides practical protection against cyber threats by alerting users to potential phishing attempts, thus safeguarding personal and sensitive information.


- Data Analytics, San Jose State University, Spring 2024

Body Movement Prediction using HARTH Data

This project focused on analyzing human activity data captured from sensors through HARTH (Human Activity Recognition through Smartphones) datasets. Initial raw sensor signals were transformed into meaningful features representing distinct physical activities. The processed data were then used to train predictive models, including XGBoost, Support Vector Machines (SVM), and Random Forest, successfully achieving a maximum accuracy of 95.01%. The robust performance of these models signifies their potential in applications such as fitness tracking, healthcare monitoring, and smart wearable devices. It illustrates the effectiveness of machine learning techniques in converting complex sensor data into actionable insights for improving human health and behavior understanding.


- Machine Learning, San Jose State University, Fall 2023

Credit Card Transaction Fraud Detection Project

This endeavor aimed at combating credit card fraud by developing advanced machine learning models designed to detect fraudulent transactions effectively. A comprehensive exploratory approach generated 1240 candidate variables, which were subsequently narrowed down to the top 20 significant variables using KS filtering and wrapper methods. By optimizing hyper-parameters, the final model achieved a commendable fraud detection rate of 55.6%. The practical impact of this model was substantial, projecting annual financial savings of approximately $21 million through early detection of fraudulent transactions. This project highlights the vital role of data analytics and machine learning in securing financial transactions, enhancing customer trust, and reducing financial losses.


- Data Mining, San Jose State University, Fall 2023

Twitter Trend Analysis for ChatGPT using Mongo Data Cloud

This analytical project centered around capturing and interpreting Twitter trends specifically related to ChatGPT. A robust data pipeline was constructed using Apache Airflow, effectively collecting and preprocessing relevant tweets in AWS. Processed data was systematically stored within MongoDB Cloud, enabling real-time analytics and trend assessment. An impact analysis dashboard was created, providing actionable insights into user sentiment, engagement, and trending topics associated with ChatGPT. This project underscored the potential of cloud-based architectures and real-time data analytics for deriving strategic insights and enhancing decision-making processes related to social media and AI-driven technologies.
